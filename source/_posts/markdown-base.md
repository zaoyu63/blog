# 标题

示例：

```
# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题

```
<!-- more -->
效果如下：

# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题

# 文字样式

示例：
```
**这是加粗的文字**
*这是倾斜的文字*`
***这是斜体加粗的文字***
~~这是加删除线的文字~~
```
效果如下：

**这是加粗的文字**

*这是倾斜的文字*

***这是斜体加粗的文字***

~~这是加删除线的文字~~


# 分隔线

示例：
```
***
*****
---
----
```

效果如下：

---
----
***
*****

# 图片

示例：

```
![飞鸟](/images/birds.jpg， "飞鸟")
```
效果如下：

![飞鸟](/images/birds.jpg "飞鸟")

# 超链接

示例：

```
[百度](https://www.baidu.com "百度")
```
效果如下：

[百度](https://www.baidu.com "百度")

# 无序列表

示例：
```
* 测试1
* 测试2
* 测试3

+ 测试1
+ 测试2
+ 测试3

- 测试1
- 测试2
- 测试3
```
效果如下：

* 测试1
* 测试2
* 测试3

+ 测试1
+ 测试2
+ 测试3

- 测试1
- 测试2
- 测试3

# 有序列表
示例：

```
1. 测试1
2. 测试2
3. 测试3
```
效果如下：

1. 测试1
2. 测试2
3. 测试3

# 表格

示例：

```
货品|数量|单价
---|:---:|---:
a| 10| $5
b| 100| $1
c| 4| $20
```

货品|数量|单价
---|:---:|---:
a| 10| $5
b| 100| $1
c| 4| $20
